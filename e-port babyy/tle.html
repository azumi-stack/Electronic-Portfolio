DISPLAY PROPERTY

I learned that the display property determines how HTML elements are shown on a webpage. The common display values are block, inline, inline-block, and none. It helps organize page layout by controlling how elements align and behave visually.

Example of block display property.

CLASSES

I learned that classes in CSS allow us to style specific elements without affecting others. They are defined with a period (.) followed by the class name and can be reused multiple times in a webpage to maintain consistency.

Example of CSS classes.

SELECTORS

I learned that CSS selectors are patterns used to target HTML elements that need styling. Examples include element selectors, class selectors, ID selectors, and group selectors.

Example of CSS selector.

DIMENSIONS

I learned that dimensions in CSS define the visibility and size of elements, using properties like width, height, max-width, and min-height. They control how large or small elements appear on the page.
